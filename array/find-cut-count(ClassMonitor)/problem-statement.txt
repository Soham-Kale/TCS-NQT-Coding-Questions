Problem 1: Class Monitor
Problem Statement:

After JEE mains, students join an engineering college. To select the class monitor, 
the HOD uses a register. Every time he meets a student with a better (smaller) 
rank than the previous best, he cuts the previous name and writes the new student’s name and rank.

Given the sequence of ranks, count how many times the name was cut.

Input Format:
rank1 rank2 rank3 ... rankn

Output Format:
Number of cuts

Example 1:
Input:
6
4 5 2 3 1 6

Output:
3

Example 2:
Input:
5
10 9 8 7 6

Output:
4
